# Quant Research

Systematic research environment for financial modeling and backtesting.

## Structure
- src/ → core modules
- notebooks/ → exploratory research
- tests/ → validation

## Setup

python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
